This Custom Unity Library contains generic extensions for the Unity 5 game engine.

The Custom Unity Library folder is where the actual library is stored.  Copy this into your project to use it.

The Demo folder has some non-generic code which uses the Custom Unity Library.  It can be used for reference or altered towards your needs.

The Gizmos folder is used by the Demo.

The Under Development folder has Custom Unity Library assets which are currently under development.  They should be viewed as reference only, and are not considered safe to use.
